# Chrome扩展安装指南

## 📦 第一步：加载扩展到Chrome

### 1. 打开Chrome扩展管理页面

在Chrome浏览器的地址栏中输入：
```
chrome://extensions/
```
然后按回车键。

**或者**通过菜单打开：
- 点击浏览器右上角的 **三个点** ⋮
- 选择 **"更多工具"**
- 选择 **"扩展程序"**

---

### 2. 开启开发者模式

在扩展程序页面的 **右上角**，找到 **"开发者模式"** 开关，将它 **打开**（变成蓝色）。

---

### 3. 加载扩展

点击左上角的 **"加载已解压的扩展程序"** 按钮。

在弹出的文件选择器中，导航到并选择以下目录：
```
/Users/lyj/Desktop/project/ChromeTools/dist
```

**重要提示**：
- ⚠️ 一定要选择 `dist` 目录（不是整个项目目录）
- ✅ dist 目录包含了构建好的扩展文件

---

### 4. 确认加载成功

加载成功后，你会在扩展列表中看到：

**Schema Editor**
- 版本：1.0.0
- 说明：DOM元素Schema查看和编辑工具
- 图标：蓝色背景的 SE 字样

---

## 🔍 第二步：找到扩展图标

### 方法1：固定到工具栏（推荐）

1. 点击浏览器地址栏右侧的 **拼图图标** 🧩（扩展程序图标）
2. 在弹出的扩展列表中，找到 **Schema Editor**
3. 点击 Schema Editor 右侧的 **图钉图标** 📌
4. 图标就会固定显示在工具栏上

### 方法2：通过扩展菜单使用

如果不想固定，也可以：
1. 点击 **拼图图标** 🧩
2. 在列表中找到 **Schema Editor**
3. 直接点击即可激活/停用

---

## ✅ 第三步：验证安装

### 检查扩展状态

在 `chrome://extensions/` 页面，确认：
- ✅ Schema Editor 的开关是 **开启** 状态（蓝色）
- ✅ 没有显示任何错误信息
- ✅ 可以看到扩展的详细信息

### 测试扩展

1. **打开测试页面**：http://localhost:8080/index.html
   
2. **激活扩展**：
   - 点击工具栏的 Schema Editor 图标（SE图标）
   - 或通过拼图菜单点击

3. **测试功能**：
   - 鼠标悬停在测试元素上
   - 应该看到蓝色高亮边框和tooltip

---

## 🔧 常见问题

### Q1: 找不到"加载已解压的扩展程序"按钮
**A:** 确保已经开启了"开发者模式"（右上角的开关）

### Q2: 加载时提示错误
**A:** 检查以下几点：
- 确保选择的是 `dist` 目录
- 确保开发服务器正在运行（dist目录有内容）
- 如果有错误信息，查看具体提示

### Q3: 扩展图标不显示
**A:** 
- 点击拼图图标 🧩 查看扩展列表
- 将Schema Editor固定到工具栏
- 或者在拼图菜单中直接使用

### Q4: 点击图标没有反应
**A:** 
- 确保在测试页面标签（http://localhost:8080）
- 查看扩展详情页的错误信息
- 按F12查看Console是否有错误

### Q5: 提示"无法加载manifest文件"
**A:** 
- 检查是否选择了正确的目录（应该是 `dist`）
- 确认 dist 目录下有 manifest.json 文件
- 重新运行 `tnpm run build` 或 `tnpm run dev`

---

## 🔄 重新加载扩展

如果修改了代码，需要重新加载扩展：

1. 在 `chrome://extensions/` 页面
2. 找到 Schema Editor
3. 点击 **刷新图标** 🔄（或点击"重新加载"按钮）
4. 刷新测试页面
